## 📋 前端功能概览

### 1. 仪表盘（Dashboard）📊
- 总小说数、今日抓取、任务成功率、失败任务统计
- 各平台小说数量分布
- 平台状态监控

### 2. 数据浏览（Novels）📚
- 小说列表展示（封面、书名、作者、平台、章节信息）
- 平台筛选（全部/刺猬猫/SF轻小说/次元姬/起点）
- 关键词搜索（书名/作者）
- 分页浏览

### 3. 平台配置（Platforms）⚙️
- 4个平台配置管理
- 启用/禁用切换
- 抓取间隔设置
- 标签管理（添加/删除）
- 手动执行爬虫
- 测试爬虫功能

### 4. 任务列表（Tasks）📝
- 爬虫任务列表
- 任务状态（运行中/成功/失败/等待中）
- 执行统计（总数/成功/失败）
- 分页浏览

### 5. 任务详情（TaskDetail）🔍
- 任务基本信息
- 执行日志查看
- 执行时长计算

### 6. 小说详情（NovelDetail）📖
- 小说完整信息
- AI概括（前3章）
- 抓取历史


## ⚠️ 常见问题

### Q1: 前端启动失败？
**A**: 确保已安装Node.js，运行 `npm install` 安装依赖

### Q2: 图片无法显示？
**A**: 使用的是随机占位图服务，可能加载较慢或偶尔失败

### Q3: 数据为空？
**A**: 检查是否使用Mock模式（USE_MOCK = true）

### Q4: API调用失败？
**A**: 确认后端已启动，检查端口是否正确（默认8080）

### Q5: 切换到真实后端后无法访问？
**A**: 检查后端是否启动，确认USE_MOCK = false
